# rnstl-lerna-workspace-example

This example shows an example of how `react-native-storybook-loader` works with lerna + yarn workspaces.

## Notes

- In lerna.json, make sure the `npmClient` is set to `yarn` and `userWorksaces` is `true`
    ```
    {
      "npmClient": "yarn",
      "useWorkspaces": true
    }
    ```
